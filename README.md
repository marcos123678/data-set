# Análise Exploratória de Dados (EDA) - Students Performance

Este repositório contém a Etapa 1 do projeto de análise exploratória de dados (EDA) utilizando o dataset `students_performance.csv`.

Foram realizadas:
- Estatísticas descritivas
- Análise de valores faltantes
- Visualização de distribuições
- Identificação de outliers
- Correlações entre variáveis

O notebook principal está em: `01_EDA.ipynb`.
